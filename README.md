Folder ini berisi latihan dasar automation menggunakan Selenium pada website publik untuk melakukan automation testing.


### ▶️ How to Run
**1. Install dependency:**
``` bash
pip install selenium
```

2. Jalankan script:
``` bash
python belajarselenium.py
```
